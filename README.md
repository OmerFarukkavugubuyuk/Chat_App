Chat App on LAN by C#

# How to Use

## Starting the Server
1. Open `ChatApp.exe`
2. Enter your name
3. Click **"Sunucu Başlat"**
4. The chat window will open — you are now hosting
5. Share your local IP address with friends (find it via `ipconfig`)

## Joining as a Client
1. Open `ChatApp.exe`
2. Enter your name
3. Click **"Bağlan"**
4. Enter the server's IP address and port (`8888` by default)
5. Click **"Bağlan"** again — you are now connected
6. The chat window will open

## Notes
- All devices must be on the same Wi-Fi or LAN network
- The server must be started before anyone can join
- Default port is `8888`
